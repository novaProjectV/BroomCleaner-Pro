**BroomCleaner Pro**

Минималистичный и прозрачный клинер для macOS. Без агрессивных скриптов, без скрытых действий, без «чёрной магии» — только то, что ты выбираешь сам.

BroomCleaner Pro аккуратно находит и подготавливает к удалению кэш, логи, временные и устаревшие файлы, показывая *полную картину* до любых изменений.

Древовидный предпросмотр:

**Приложение → Тип файла → Полный путь → Размер**

Максимальный контроль над каждым файлом: выбор, исключения, защита от случайных действий.

Удаление выполняется **только через Корзину** с безопасным окном Undo на 15 минут — если передумал, всё можно легко восстановить.

Внутри: умное прогресс-кольцо с ETA, статистика освобождённого места за месяц и гибкая система исключений (Whitelist).

⚠️ Для корректной работы требуется Full Disk Access:

System Settings → Privacy & Security → Full Disk Access → включить для BroomCleaner Pro

⸻

### Возможности

• **Один клик — полный скан.** Автоматический поиск кэшей, логов, временных и устаревших файлов.

• **Древовидный предпросмотр.** Удобная структура: App → Тип данных → Путь → Размер + чекбоксы.

• **Безопасное удаление.** Все файлы отправляются в Корзину через FileManager.trashItem.

• **Undo на 15 минут.** Мгновенный возврат удалённого, без риска потери.

• **Умная кнопка прогресса.** Анимированное кольцо, ETA и «пружинка» на завершении.

• **Режимы риска + давность.** Настройка глубины очистки и фильтрация по возрасту файлов.

• **Whitelist.** Исключения для конкретных путей и каталогов.

• **Статистика.** Отчёт по освобождённому месту за месяц и результат последнего запуска.

• **Доступность.** Полная поддержка VoiceOver через accessibilityLabel / accessibilityValue.

• **Прозрачность.** Понятный лог: что удалено, сколько весило, были ли ошибки.

• **MacPulse.** Детали в релизах версии 1.1.0.

⸻

### Как это работает

1. **Сканирование** — приложение находит кандидатов с учётом риска, давности и исключений.  
2. **Предпросмотр** — ты просматриваешь дерево и вручную управляешь каждым файлом.  
3. **Удаление** — выбранные элементы отправляются в Корзину + активируется окно Undo.  
4. **Завершение** — обновляется статистика, ETA и системный статус.
